Degu Desktop

Windowsのタスクバー上で、ピクセル風のデグーがちょこちょこ歩き回るデスクトップペットです。 キーボード入力やクリックに反応し、採食、食べる、掘る、毛づくろい、自然な左右ターン、入力中だけ回る回し車に対応しています。

Windows版 x64 Windows版 x86 Releases
バージョン dev-local 更新日 2026-06-20 コミット local
Windows 10/11、x64 / x86向け。ZIPを展開して DeguDesktop.exe を起動してください。

タスクバー上のペット

透明な最前面レイヤーで、タスクバー付近にデグーがいるように見せます。

最大10匹

トレイメニューや設定画面から、1、2、3、5、10匹を切り替えられます。

設定画面

毛色の決め方、1匹ごとの色、速度、入力モード、回し車、言語、自然な左右ターンを調整できます。

自然な動き

歩く、止まる、走る、左右ターンをつなぎ、急な反転が目立たないようにしています。

採食と遊び

小さな牧草、枝、種のようなアイテムを嗅ぐ、食べる、掘る、かじる、運ぶ動きがあります。

入力中の回し車

キーボード反応モードでは、入力している間だけ回し車が動きます。

クリック反応

デグーをクリックすると、ハートなどの小さな吹き出しが出ます。デスクトップへのクリックは通します。

毛色バリエーション

アグーチ、ブラック、ブルーグレー、グレー、クリーム、チョコレート、パイド系の毛色を収録しています。

配布ビルド

このページのダウンロードリンクから、Windows向けZIPを取得できます。

English

Degu Desktop is a small Windows desktop pet app. Pixel-art degus walk above the taskbar, react to typing and clicks, forage, eat, dig, groom, turn smoothly, and run inside a wheel while you type. Download the Windows x64 or x86 ZIP, unzip it, and run DeguDesktop.exe.
