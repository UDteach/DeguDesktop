Degu Desktop

Windowsのタスクバー上やMacのDock上で、ピクセル風のデグーがちょこちょこ歩き回るデスクトップペットです。 キーボード入力やクリックに反応し、Windows版では採食、食べる、掘る、毛づくろい、自然な左右ターン、入力中だけ回る回し車にも対応しています。

Windows版 x64 Windows版 x86 Mac版 Apple Silicon Mac版 Intel

Mac版はmacOS 12 Monterey以降対応です。M1 / M2 / M3 / M4などはApple Silicon版、Intel MacはIntel版を選んでください。

バージョン dev-local 更新日 2026-06-20 コミット local
Windows 10/11はZIPを展開して DeguDesktop.exe を起動してください。 Mac版はZIPを展開して DeguDesktop.app をアプリケーションフォルダに入れてください。

タスクバー上のペット

透明な最前面レイヤーで、タスクバー付近にデグーがいるように見せます。

最大10匹

設定画面から1〜10匹を切り替えられます。Mac版のメニューバーにも1〜10匹の表示数を用意しています。

設定画面

毛色、1匹ごとの色、名前、速度、入力モード、回し車を調整できます。Windows版では言語やアップデート確認も扱えます。

自然な動き

歩く、止まる、走る、左右ターンをつなぎ、急な反転が目立たないようにしています。

Mac版

Mac版はメニューバーのデグーアイコンから設定画面を開けます。速さ、1〜10匹の表示数、毛色、名前、モード、回し車を操作でき、Dock上の下端でデグーがうろちょろします。

採食と遊び

Windows版では、小さな牧草、枝、種のようなアイテムを嗅ぐ、食べる、掘る、かじる、運ぶ動きがあります。

入力中の回し車

Windows版のキーボード反応モードでは、入力している間だけ回し車が動きます。

クリック反応

Windows版とMac版では、デグーをクリックするとハートなどの小さな吹き出しが出ます。デスクトップへのクリックは通します。

毛色バリエーション

アグーチ、ブラック、ブルーグレー、グレー、クリーム、チョコレート、パイド系の毛色を収録しています。

配布ビルド

このページのダウンロードリンクからWindows向けZIPを取得できます。Mac版はGitHub Releasesで配布します。現在のMac版はmacOS 12 Monterey以降対応です。

English

Degu Desktop is a small desktop pet app for Windows and macOS. Pixel-art degus walk above the Windows taskbar or along the bottom edge above the Mac Dock and react to typing and clicks. The Windows build also supports foraging, eating, digging, grooming, smooth turns, and wheel running while you type. Download the Windows x64 or x86 ZIP from this page, or get the macOS ZIP from GitHub Releases.
