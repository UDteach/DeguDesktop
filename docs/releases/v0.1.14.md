# DeguDesktop v0.1.14

Date: 2026-06-29

This release fixes Windows multi-monitor rendering on mixed-height and mixed-DPI display layouts.

## Download

- `DeguDesktop-windows-amd64.zip` for Windows 10/11 x64
- `DeguDesktop-windows-386.zip` for Windows 10/11 x86
- `SHA256SUMS.txt` for Windows package and packaged EXE verification

Each Windows ZIP includes:

- `DeguDesktop.exe`
- `README.md`
- `SECURITY.txt`

## Windows Changes

- Split the transparent pet overlay into per-monitor layered-window segments.
- Added per-monitor DPI scaling for overlay height, vertical offset, wheel placement, forage props, pet sprites, click hit testing, name labels, and reaction bubbles.
- Fixed layouts where pets could align to the wrong monitor bottom or appear missing on a secondary display when monitors have different heights or DPI scaling.
- Kept the existing monitor/span selector and walking-range behavior while placing each visible segment on its own monitor bottom.
- Added cleanup for extra overlay windows created for multi-monitor spans.

## Known Status

- Authenticode signing depends on configuring Azure Artifact Signing or a valid signing certificate.
- Unsigned Windows builds may still show Microsoft Defender SmartScreen or antivirus reputation warnings until signing, reputation, or allowlisting catches up.
- macOS download links remain on the existing `v0.1.9` artifacts until a separate macOS release sync is built and uploaded.
